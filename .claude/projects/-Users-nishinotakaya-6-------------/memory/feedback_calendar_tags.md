---
name: カレンダー詳細にも投稿タグ表示
description: カレンダーの日付クリック→イベント詳細にも、一覧カードと同じ投稿状況タグを常に表示する
type: feedback
---

カレンダービューの詳細（日付クリック→イベント一覧）にも、一覧カード（ItemCard）と同じ投稿状況タグ（✅📝❌💰）を表示すること。

**Why:** 2026-04-14のセッションで3回以上指摘された。CalendarView と ItemCard で表示内容が一致していないとユーザーが混乱する。「一覧にはあるのにカレンダーにない」は致命的。

**How to apply:**
- CalendarView内のイベント詳細表示には必ず投稿タグを含める
- ItemCardと同じステータス色・アイコン・クリック遷移を実装
- 新機能をItemCardに追加したら、CalendarEventCardにも同等の表示を追加すること
