---
name: 画面切替時のリロード不要化
description: フォルダ選択・カレンダー↔一覧切替時にリロードなしでデータ表示されること
type: feedback
---

画面切替（フォルダ選択、カレンダー→一覧、ログイン後）でリロードしないとデータが表示されない問題が繰り返し発生。

**Why:** 2026-04-11〜14で4回以上指摘。ユーザーが毎回リロードを強いられるのはUXとして最悪。

**How to apply:**
- 画面切替時（setShowCalendar, handleSelectFolder, ログイン後）は必ず `loadAll()` を呼ぶ
- React の state 更新後にデータが表示されるか、ブラウザで実際に確認してからデプロイ
- 特にviewerロールでの初回ログイン後の表示を必ずテスト
