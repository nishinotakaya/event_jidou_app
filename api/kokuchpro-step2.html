イベント登録

  • マイページ
  • イベント登録

詳しい使い方はこちら

イベントの情報を入力して「登録する」ボタンをクリックしてください。
こちらの情報は、イベント公開まではイベント管理ページにて変更することが可能です。
本日のイベント登録数が制限（1日最大3件まで）に達しました。
明日以降にイベント登録を行ってください。
※プレミアム会員にご加入いただくことで、1日に登録可能なイベント数が、20件にアップします！
プレミアム会員に申込む

基本情報

オフライン

無料イベント

必須イベントの名前を入力してください
必須イベント一覧に表示されるイベントの概要を80文字以内で入力してください

から

必須終了日を省略した場合は開始日と同じになります
必須イベントで募集する人数を指定してください（最大: 1,000人）

から

指定しない場合は、公開日からイベント開始日までとなります

イベント内容

ファイル選択   変更   クリア
JPG、GIF、PNG画像がアップロードできます（5Mbyteまで）
※横幅660ピクセルにリサイズされます（高さは縦横比を保持）

イベントの案内文を入力してください

p
SCRIPT, IFRAME, LINKタグは使用できません。 0 / 65535 byte

開催場所

必須開催場所の会場名を入力してください
会場のホームページがある場合はURLを入力してください
開催場所の住所を入力してください
開催場所の住所が不明瞭なイベントは検索結果に表示されません。
会場となる開催場所の住所を正しく入力してください。
開催場所のビル名・階数を入力してください

マーカーをドラッグして開催場所の位置を指定してください

イベント事務局

事務局名は窓口となる会社名や団体名・コミュニティ名などを入力してください。指定しない場合はイベント名が事務局名になります
必須事務局に連絡がとれる電話番号（ハイフンを含める）を入力してください。（イベントの参加者にだけ公開されます）
必須申込受付時や問合受付時に送信されるメールアドレスを入力してください
主催しているグループを選択してください。
グループの登録はこちらから行えます。

イベントチケット

チケット名 金額(税込) 販売枚数
イベントチケットを追加

懇親会チケット

チケット名 金額(税込) 販売枚数
懇親会チケットを追加

※イベント登録後に、イベント管理画面で「公開する」を行うとイベントが公開されます。
※イベントのオプション情報については、イベント公開前に管理画面の「基本情報設定」で変更することができます。
